# IoT Smart Helmet 🚧

## What this project does
This project improves construction worker safety by detecting:
- Falls
- Gas exposure
- Helmet wearing status

## Components used
- ESP8266
- MPU6050
- MQ135 gas sensor
- IR sensor

## How it works
Sensors collect data → ESP8266 processes → sends alert to mobile
